OIKOMIとは

大学生向けの次世代タスク管理アプリ。残り時間ではなく、あなたの心理的な「ヤバさ」で課題を管理します。

🎯 3ステップで始まる

所要時間を入力 → やりたくなさを選ぶ → ヤバさ指数が計算される

🔔 5段階の状態表示

平和 → そのうちやろ → そろそろ現実 → 逃げ場なし → 戦争

🔒 完全プライベート

すべてのデータはあなたのiPhoneにだけ保存。誰にも見られません。
